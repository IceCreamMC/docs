# Installation
Are you ready to install the best fork of purpur there is.

Here are the steps
1. Download IceCream From [Github Releases](https://www.github.com/IceCreamMC/IceCream/releases/latest)
2. After installing upload it to your Minecraft Server and name it to what your startup command is!
3. After uploading start up your server and you are done!
